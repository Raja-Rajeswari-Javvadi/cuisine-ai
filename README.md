# 🍽️ CuisineAI

### Transform Ingredients Into Extraordinary Meals

CuisineAI is a modern AI-powered culinary platform that helps users generate personalized recipes, discover dishes from available pantry ingredients, plan weekly meals, and interact with an intelligent cooking assistant powered by Google Gemini AI.

---

## ✨ Features

### 🤖 AI Recipe Generator
Generate detailed and personalized recipes based on:

- Dish name or theme
- Cuisine preference
- Dietary restrictions
- Available ingredients

### 🥗 Pantry Matcher
Enter ingredients available in your kitchen and instantly discover recipes you can prepare.

### 📅 Weekly Meal Planner
Generate balanced weekly meal plans based on:

- Calorie goals
- Dietary preferences
- Health objectives

### 💬 AI Chef Assistant
Interactive AI chatbot that helps users with:

- Cooking guidance
- Ingredient substitutions
- Recipe recommendations
- Kitchen tips and tricks

### 🌐 Multilingual Support
Supports:

- English
- Telugu

### ❤️ Recipe Library
Save and revisit your favorite AI-generated recipes.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/cuisine-ai.git
cd cuisine-ai
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
NODE_ENV=development
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:3000
```

---

## 🏗️ Production Build

Build the application:

```bash
npm run build
```

Start production server:

```bash
npm start
```
